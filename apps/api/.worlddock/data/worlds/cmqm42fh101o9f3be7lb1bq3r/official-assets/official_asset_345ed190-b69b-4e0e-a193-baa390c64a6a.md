# 重力：宇宙纪元的底层物理规则

## 一、技术边界：无人工重力

本世界严格遵守硬科幻底线——**不存在引力场发生器、重力板、反重力装置**等虚构科技。人类仅能通过两种方式获得人造重力环境：

1. **旋转离心力**：通过飞船/空间站的旋转结构产生模拟重力。
2. **飞船加速**：在持续加速（如聚变引擎推进）阶段，惯性力模拟重力。

> **核心约束**：任何旋转结构产生的等效重力加速度 **≤ 1/6g**（约等于月球表面重力）。超过此阈值时，结构材料强度无法承受所需的旋转速度和半径组合，或建造成本超出了经济可行性。

## 二、旋转重力的工程规范

| 参数 | 标准值 | 说明 |
|------|--------|------|
| 最小旋转半径 | ≥ 100 m | 小于此半径时科里奥利效应导致严重眩晕 |
| 最大转速 | ≤ 4 rpm | 人体舒适上限 |
| 最大等效重力 | 1/6 g | 结构强度与经济性共同决定的工程天花板 |
| 推荐舒适重力 | 0.16–0.2 g | 常用目标（类比月球至略高于月球） |

### 技术细节

- 在给定旋转半径 R 和转速 ω 下，等效重力加速度 g_eq = ω²R。
- 要达到 1/6g（约 1.63 m/s²），若半径 = 100 m，则需转速 ≈ 2.16 rpm（在舒适范围内）。
- 若半径缩小到 50 m，需转速 ≈ 3.05 rpm，仍在可接受范围，但科里奥利效应开始明显。
- 若要达到 1g（9.81 m/s²），以 100 m 半径计算需转速 ≈ 5.3 rpm——已超出 4 rpm 的舒适上限，且结构载荷过高。

## 三、天然重力环境分级

| 等级 | 范围 | 典型环境 | 人类适应状态 |
|------|------|----------|-------------|
| 高重力 | > 1.2 g | 无（天然，地球是上限） | 不适，心血管负担大 |
| 标准重力 | 1.0 g | 地球表面 | 人类基准 |
| 中低重力 | 0.3–0.6 g | 火星（0.38g） | 可中期居住，需康复训练 |
| 低重力 | 0.1–0.2 g | 月球（0.165g），旋转空间站 | 长期居住导致不可逆生理变化 |
| 微重力 | < 0.01 g | 太空船内部（非旋转段），小行星表面 | 必须严格限时，否则严重健康损害 |
| 近零重力 | ≈ 0 g | 自由轨道，深空航行 | 只能在专门设计的短期任务中暴露 |

## 四、生理影响：不可逆的「地-空分化」

### 4.1 低重力综合征（Low-G Syndrome）

长期生活在 ≤ 1/6g 环境中的个体，会出现以下适应性变化：

- **骨骼脱钙**：骨密度每十年下降约 10–15%，在 ≤ 0.2g 环境中不可逆转
- **肌肉萎缩**：抗重力肌群（脊柱竖脊肌、股四头肌、小腿肌群）退化至基准的 50–60%
- **心血管重塑**：心脏缩小、血容量下降、血压调节机制重置为低重力模式
- **前庭系统重校准**：平衡感、空间定向能力完全适应低重力环境

### 4.2 太空一代（Space-born）

在 ≤ 1/6g 环境中从出生到成年的个体，**将永远无法返回地球表面**：

- 骨骼峰值密度仅为地球正常人的 40–50%
- 心肌力量不足以对抗 1g 重力泵血
- 肺部无法在 1g 环境下维持正常通气分布
- 任何试图降落地球的行为都将在数小时内引发心力衰竭或肺水肿

### 4.3 返回地球的「适配税」

地球短期访问者（≤ 2 年低重力暴露）需经历：
- **康复周期**：每在低重力环境停留 1 个月，需要约 1 周的高强度康复训练
- **辅助设备**：外骨骼助力套装为标配，用于补充退化肌群
- **医疗监护**：重返地球后前 72 小时强制监护

## 五、社会文化影响

### 5.1 重力阶层

基于所在环境的长期重力水平，人类社会事实上分化为三个阶层：

| 阶层 | 居住环境 | 特征 |
|------|---------|------|
| **地表人**（地球本土） | 1g，天然土壤 | 政治权力中心，拥有返回地球的权利 |
| **轨道人**（太空站/大型飞船） | 0.16–0.2g（旋转段） | 中产阶级/技术精英，需轮班锻炼维持骨骼 |
| **深空人**（小行星带/月球殖民地） | ≤ 0.165g 或微重力 | 底层劳工、矿工，几乎不可逆的低重力适应 |

### 5.2 重力歧视与政治矛盾

- 地表人视轨道人为"软骨病"，视深空人为"异类"
- 深空人将返回地球视为无法实现的梦想，产生"被放逐"的集体心态
- 殖民地独立运动的核心诉求之一：**取消不可逆返地的结构性压迫**
- 人权争议：是否允许新生儿在低重力环境中出生？这是否构成"非自愿的身体改造"？

### 5.3 文化符号

- **重力节**：太空殖民地每年在模拟重力最高值日举行传统仪式，纪念与地球的联系
- **重力信仰**：部分边缘宗教团体崇拜1g重力为"神赐之重"，将低重力视为堕落

## 六、伦理困境：重力的政治化

重力不再只是物理学概念，而是**最根本的社会权力工具**：

1. **移民控制**：地球政府可"不可逆地"将罪犯或政治异见者流放到低重力殖民地
2. **生育限制**：是否应禁止在 ≤ 0.2g 环境下怀孕？这侵犯生育权，还是保护后代？
3. **殖民地地位**：永远无法返回地球的人群，是否应拥有完全自治权？
4. **医疗资源分配**：人造重力舱（旋转舱段）的成本极高，谁有资格享有？
