# 重力

## 一、重力技术状态

### 1.1 核心原则
宇宙纪元中，人类**未掌握任何形式的引力场发生器或反重力装置**。所有在太空中感受到的"重力"均来自以下两种物理机制：

| 机制 | 原理 | 适用场景 |
|------|------|----------|
| **旋转离心力** | 通过空间站/飞船自旋转，利用离心力模拟重力 | 长期驻留空间站、大型殖民船、环形轨道城 |
| **加速度推力** | 飞船持续加速（或减速）时，惯性力等效重力 | 星际航行期间的巡航阶段、舰船变轨机动 |

### 1.2 旋转重力的工程约束
由于结构强度和材料疲劳限制，**通过旋转模拟的重力上限为 1/6g**（约等于月球表面重力）。原因如下：

1. **结构强度不足**：要模拟 1g 重力，旋转半径和转速的组合会对飞船/空间站骨架产生巨大的离心应力。当前材料工艺无法在成本和经济性可接受的范围内建造能承受 1g 旋转的大型结构。
2. **科里奥利效应阈限**：为避免人体不适，旋转半径需 ≥ 100m，转速 ≤ 4rpm。在此约束下，若要达到 1g，所需半径过大，超出工程可行范围。
3. **能源与经济成本**：旋转结构需要持续的动量维护和姿态调节能耗，1/6g 是聚变能源经济性可支撑的平衡点。

### 1.3 舒适度标准
- **舒适区**：旋转半径 ≥ 100m，转速 ≤ 2rpm，模拟重力 0.1g~0.16g
- **可耐受区**：旋转半径 ≥ 50m，转速 ≤ 4rpm，模拟重力 0.05g~0.16g
- **不适区**：转速 > 4rpm，科里奥利效应导致前庭系统紊乱，空间定向障碍

## 二、天然重力分级

宇宙纪元将人类可遭遇的重力环境分为以下等级：

| 等级 | 重力范围 | 典型环境 | 人体状态 |
|------|----------|----------|----------|
| **1G标准区** | 0.8g~1.2g | 地球表面 | 人类基准生理环境 |
| **中低重力区** | 0.3g~0.8g | 火星(0.38g) | 肌力衰减但可逆；返回地球需6~12个月康复 |
| **低重力区** | 0.05g~0.3g | 月球(0.165g) | 长期居住导致骨密度下降30%~50%，肌肉萎缩 |
| **微重力区** | 0.001g~0.05g | 太空站内部、小行星表面 | 体液上移、骨质疏松、前庭紊乱 |
| **近零重力区** | <0.001g | 深空自由飞行、小质量天体 | 需主动约束与锻炼对抗完全失重 |
| **高重力区** | >1.2g | 特定加速段、巨型气态行星 | 需加压抗荷服，短期暴露 |

## 三、生理影响：不可逆的"地-空分化"

### 3.1 长期低重力适应症候群
在 ≤ 1/6g 环境中长期生活（>2年），人体会发生以下不可逆或半可逆变化：

- **骨骼密度重置**：骨钙流失速度约每月1%~1.5%，5年后达到适应 1/6g 的稳态（约为地球常人的 40%~60%）
- **肌肉退化**：抗重力肌群（小腿、背部、核心）萎缩60%以上
- **心血管重构**：心脏缩小、血容量下降，回到 1g 时无法对抗重力泵血，出现体位性低血压晕厥
- **前庭系统退化**：空间定向能力改变，返回地球后出现严重晕眩

### 3.2 "太空一代"——不可逆的生理禁锢
**在 ≤ 1/6g 环境中出生长大的人，其骨骼和肌肉系统以 1/6g 为基准发育，无法适应地球 1g 环境。** 具体表现：
- 骨密度峰值仅为地球正常人 50%~60%
- 心脏泵血能力不足以在 1g 下维持脑部供血
- 肺活量和膈肌力量不足以对抗地球重力呼吸
- 即使长期康复训练，也无法完全恢复至地球基准

**结论：** 在太空出生的"太空一代"将永远无法返回地球表面。他们是被重力法律性地、生物性地永久隔离在太空的族群。

### 3.3 短期驻留者的康复成本
地球人前往太空驻留后返回，需经历"重力康复期"：
- 驻留 ≤ 6个月：康复期 1:1（驻留1个月康复1个月）
- 驻留 6~24个月：康复期 1:2
- 驻留 > 24个月：部分损伤不可逆

康复治疗需在专门的"重力梯度疗养舱"中进行，费用高昂。

## 四、社会影响：由重力划分的阶层

### 4.1 三种"重力人"
重力不可跨越的生理壁垒直接催生了人类社会的新阶层划分：

| 阶层 | 定义 | 特征 |
|------|------|------|
| **地表人 (Tellurian)** | 出生且居住在地球 1g 环境 | 最高生理基准；可自由选择前往太空（但需承担康复成本） |
| **低重力人 (Hypograv)** | 在 ≤ 1/6g 太空环境中出生或长期居住者 | 无法返回地球；身体结构适应低重力；太空殖民地本土居民 |
| **过渡人 (Transients)** | 往返于地球与太空的短期驻留者 | 在两界间反复适应；健康损耗大；多为精英技术人员或富商 |

### 4.2 "重力税"
地球政府/殖民管理机构对从太空返回地球的人员征收的隐性成本：
- 强制6~12个月重力康复疗程（费用由个人或派遣机构承担）
- 康复期间禁止从事高负荷工作
- 每公斤返回人员需配给等重的压舱物（增加运输成本）

### 4.3 伦理困境
- 太空殖民地是否是被重力"永久流放"的人群？
- 地球政府是否应当对太空殖民者的健康损害负责？
- 低重力人是否有权获得地球的医疗资源（明知他们无法承受 1g 环境）？
- 是否应该研发强制性重力康复技术（如药物增强骨密度），干预人体自然适应？

## 五、工程与设计规范的衍生约束

重力规则对太空设施设计产生的连锁约束：

| 领域 | 约束内容 |
|------|----------|
| **飞船设计** | 环形/轮形旋转段为标配；轴向加速度段为辅助；非旋转段（核心舱）为微重力工作区 |
| **城市设计** | 太空城需分层设计：微重力工业区→低重力居住区→旋转重力农业区（0.1~0.16g） |
| **医疗设施** | 所有太空殖民地必须配备重力梯度康复舱和低重力手术台 |
| **体育与训练** | 太空住民每日需进行抗阻训练对抗骨密度流失；重力笼（人造重载训练设备）为标配 |
| **生育政策** | 部分殖民地禁止在低重力环境下怀孕，要求胚胎发育在旋转重力段进行 |
| **货物运输** | 压舱物配比：每吨上行货物需配0.3吨下行压舱（维持旋转平衡） |
