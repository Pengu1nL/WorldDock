# Beta 模板库

WorldDock Cloud Alpha 不包含模板库。Alpha 阶段的产品重心是让个人世界观创作者完成云端建档、结构化整理、AI 协作、发布与导出闭环，而不是提供可复用题材模板或模板市场。

## Alpha 不提供

- 模板列表页：不提供可浏览、筛选、搜索或收藏的模板目录。
- 模板详情页：不提供模板预览、版本说明、作者介绍、适用题材或使用案例页面。
- 模板驱动新用户引导：新用户引导不以选择模板作为必经路径，也不承诺一键生成完整世界。
- 官方题材模板：不发布官方奇幻、科幻、都市、跑团、游戏设定等题材模板。
- 社区投稿模板：不开放用户提交、审核、展示或复用社区模板。
- 模板市场、授权与收入分成：不提供模板购买、售卖、授权条款、付费下载、分成结算或创作者收益记录。

## Alpha 表述边界

可以描述 WorldDock 支持创作者从空白世界开始沉淀设定、资产和发布快照；不应暗示 Alpha 已经内置模板库、官方模板、社区模板或模板市场。任何示例世界、演示数据或截图都只能作为产品演示素材，不代表可用模板资产。

## Beta 再评估

Beta 阶段再评估模板库是否进入范围，评估项包括模板授权、作者署名、派生关系、质量审核、版本兼容、侵权处理和下架流程。只有在这些规则明确后，才可以对外承诺模板分发或模板商业化能力。
