# 定价

WorldDock Cloud Alpha 免费试用。Alpha 阶段可以展示定价方向和候选套餐，但不处理真实付款，也不建立任何真实订阅、发票、税务或退款流程。

## Alpha 定价页口径

- 定价页可以明确写明 Alpha 免费试用。
- 定价页可以展示 Beta 候选套餐，用于收集用户预期和验证包装方式。
- 所有套餐按钮应表达为加入候补、了解 Beta、联系团队或类似非付款动作。
- 页面文案必须说明当前不会扣款，不会创建订阅，也不会要求用户完成真实结账。

## Alpha 不处理真实付款

Alpha 不处理真实付款，不创建 Stripe checkout 或 customer portal，不同步订阅、发票、税务、退款或付款状态。产品、接口、后台任务和文档都不应暗示用户已经可以购买、升级、续费、取消订阅或下载正式发票。

## Beta 候选套餐

- Creator：面向个人创作者，强调更多世界、更多 AI 协作额度、发布与导出能力。
- Studio：面向小型创作工作室或项目组，未来可能覆盖更高用量、更多协作空间和优先支持。
- Team：面向需要团队治理、权限、审计和组织级结算的用户，属于 Beta 或更晚阶段候选方向。

这些套餐名称只代表候选包装，不代表 Alpha 已经具备付费账户、团队权限或组织结算。

## 禁止事项

Alpha 禁止接入 Stripe secret、checkout session、customer portal、webhook、subscription 或 invoice。也禁止在 Alpha 中实现真实支付链路、真实订阅状态同步、税务计算、退款处理、付款失败恢复或付费权益开关。
