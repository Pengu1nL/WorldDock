# 产品定位

WorldDock Cloud Alpha 面向个人世界观创作者，帮助小说、游戏、跑团、漫画、影视设定和独立创作项目的作者，把分散的世界观资料整理成可维护、可发布、可导出的云端世界资产。

## 目标用户

- 正在长期维护原创世界观的个人创作者。
- 需要管理角色、地点、组织、事件、规则、时间线和发布说明的作者。
- 希望 AI 帮忙整理、补全、解释和迭代设定，但仍保留创作主导权的用户。
- 希望公开展示部分设定，同时保留私有草稿和未发布资产的创作者。
- 需要把世界观数据导出到其他工具、仓库或备份流程中的用户。

## Alpha 主张

- 云端优先：Alpha 优先提供 WorldDock Cloud 工作流，降低本地环境、同步和部署负担。
- 结构化世界资产：把世界观内容沉淀为可管理的实体、关系、发布快照和世界包，而不是只保存散乱文本。
- 可解释 AI 协作：AI 输出需要围绕已有世界资产给出上下文、依据和可追踪的修改建议，避免黑箱式替作者定稿。
- 可携带数据：用户可以导出世界包和账户相关数据，为备份、迁移和二次处理保留空间。
- 社区轻闭环：Alpha 支持以发布快照、Fork、反馈和通知形成轻量社区闭环，不把社区运营、审核后台或商业市场作为首发重点。

## Alpha 不承诺

Alpha 不承诺本地部署、模板库、真实支付、邮件投递、管理后台或团队权限。对外叙述应避免把这些 Beta 或更晚阶段能力包装成当前可用功能。

## 对外口径

Alpha 是面向个人创作者的云端世界观工作台。它的价值在于把创作资料变成结构化、可解释、可发布、可携带的世界资产，并让公开发布与社区反馈形成最小闭环。
